## [0.1.0] - 2025-01-07

### Added
- Initial release
- `ollama-prompt` tool for sending prompts to Ollama models
- `/ollama-models` command to list available models
- `/ollama-running` command to list loaded models
- `/ollama-pull` command to pull models from library
- `/ollama-prompt` command for quick prompting
- `/ollama-show` command to display model information
- `/ollama-copy` command to copy models
- `/ollama-delete` command to delete models
- `/ollama-embed` command for embeddings generation
- `/ollama-version` command to check Ollama version
- `/ollama-chat` command for chat completion
- Configuration file support with environment variable override
- Integration test infrastructure
- Unit tests for configuration
